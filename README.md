# 100Hires Portfolio Project - AI Tools Setup

This repository documents the completion of the first step for the 100Hires selection process. It outlines the tools installed, the process followed, and how potential setup challenges were proactively handled.

## 1. Tools Installed
•⁠  ⁠*Cursor IDE*: Successfully configured as the core development environment on macOS.
•⁠  ⁠*Claude Code for VS Code*: Installed and authenticated within Cursor to leverage Anthropic's advanced contextual modeling.
•⁠  ⁠*Codex Add-on*: Integrated and verified via OpenAI authentication for structural assistance.

## 2. Steps Completed
•⁠  ⁠*Environment Configuration*: Downloaded and deployed the Cursor platform onto a MacBook.
•⁠  ⁠*Extension Management*: Searched, verified, and successfully initialized both the ⁠ Claude Code ⁠ and ⁠ Codex ⁠ plugins from the marketplace.
•⁠  ⁠*Version Control Setup*: Established a public GitHub repository titled ⁠ 100hires-portfolio ⁠ to house this report and future tracking documentation.

## 3. Issues Encountered & Solutions
•⁠  ⁠*Issue - Interface Discrepancy*: During initial setup, the default interface loaded directly into the "Cursor Agents" panel instead of the standard terminal editor interface.
•⁠  ⁠*Solution*: Diagnosed the configuration layout promptly. Resolved the bottleneck by utilizing the operational file menu commands to bypass the onboarding agent stream (⁠ Skip ⁠) and launched a clean ⁠ Open Editor Window ⁠ to reveal the extension marketplace deployment area.
